# MOMOCrash-Cheat

A cheat tool for the Steam version of MOMOCrash (MomottoCrash)

Reference: [![Omega172](https://github.com/Omega172.png?size=20)](https://github.com/Omega172) [Omega172](https://github.com/Omega172)

[中文说明](README_CN.md)

## Hotkeys
- `Insert` - Show/Hide menu
- `End` - Unload cheat

## Notice
If the menu is not showing, run the game in windowed mode and expand the window before injecting!

## Features

### Game Cheats
- Auto Play
- No Miss
- All Perfect
- Infinite Gauge
- Force Full Combo
- Skip Affinity Animation
- Skip Result
- Set Gauge Value

### Save Management
- Save Game
- Reset Story
- Reset All
- Character Affinity Modification (Daia / Hikaru / Seika)
- Max Affinity (one-click)

### Game State / Modifier
- Real-time display: Score, Combo, Max Combo, Gauge
- Judgment statistics (Perfect / Good / Bad / Miss)
- Set Score
- Set Combo
- Custom Gauge Values (Perfect / Good / Bad)
